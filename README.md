**PickPipe意在将Termux中的dpkg包管理器进行简化**

## 🏁快速开始
- 请先确保你的Termux已经安装了curl:
`pkg i curl`
- 置备好了curl，你就可以开始了:
`bash -c "$(curl -fsSL https://raw.githubusercontent.com/EdgeS5352/PickPipe/master/fast-up.sh)"`

### 📋Usage：pipe [命令] [参数]
命令: 

- d    安装软件包
- r    移除软件包
- s    查找软件包
- h    命令及参数
- p    拉取新版包
- l    列出软件包
- Uy   更新软件包
- Ud   升级软件包
