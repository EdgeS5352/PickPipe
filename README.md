# PickPipe
**PackPipe意在将dpkg包管理器进行简化**

## 安装
**下载git:**
`<pkg in git>`

**用git来将PackPipe的全部文件clone下来:**
`<git clone https://github.com/EdgeS5352/PickPipe>`

**cd进入到clone好的PickPipe所属目录中:**
`<cd PickPipe/>`

**找到安装脚本"set-up.sh"，用bash运行:**
`<bash set-up.sh>`

-**至此，PackPipe即安装完毕**

### Usage：pipe [命令] [参数]
命令: 

- d    安装软件包
- r    移除软件包
- s    查找软件包
- h    命令及参数
- Uy   更新软件包
- Ud   升级软件包
